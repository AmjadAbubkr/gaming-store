# GitHub Pages Setup

To publish the legal pages for Google Play:

1. Open the GitHub repository settings.
2. Go to **Pages**.
3. Set the source to deploy from the `main` branch `/docs` folder.
4. Save.

Once enabled, these URLs should become available:

- `https://amjadabubkr.github.io/gaming-store/privacy-policy.html`
- `https://amjadabubkr.github.io/gaming-store/account-deletion.html`
